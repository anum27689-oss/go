# **App Name**: Islamic Daily Companion

## Core Features:

- Prayer Times Display: Allows users to select their city and view daily prayer times in a clean card layout, with daily auto-refresh based on geographic location.
- Tasbeeh Counter: A digital counter with a large tap button for easy incrementing, a reset option, and automatic saving of the last count locally.
- Qibla Direction Compass: Provides a compass-based arrow indicating the Qibla direction, showing degrees for precise orientation using device sensors.
- Quran Audio Player: Displays a list of Surahs, enabling users to tap and stream audio recitations with basic play/pause controls for online listening.
- Islamic Calendar: Shows the current Hijri date and an Islamic months list within a simple, clear calendar layout.
- App Settings and Preferences: Includes a language selector for Urdu/English, a dark mode toggle for UI theme, and links to the privacy policy page and app rating function.

## Style Guidelines:

- The primary color palette will be based on a clean Islamic green theme, evoking tranquility and nature.
- The app background will feature subtle, thematic mosque-inspired elements, integrated with a soft Islamic aesthetic to create an immersive and serene environment.
- The home screen utilizes a grid layout for the main feature buttons, ensuring intuitive and easy navigation.
- Crescent moon icons are consistently used throughout the app for branding and key Islamic features.
- UI elements such as buttons will feature rounded corners and soft shadows for a modern, friendly, and accessible look.
- Smooth animations are applied to screen transitions, button interactions, and content loading to enhance user engagement without distraction.
